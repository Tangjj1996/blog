## 项目介绍

“择趣转盘”是一款为广大用户提供方便、快捷的大转盘功能，它主要解决三大场景：

1. 今天不知道吃什么，想要随机点一份套餐
2. 今天不知道做什么运动，想要随机做一下运动
3. 组局吃饭，想要在群聊里随机抽取一位“幸运观众”买单

我们精心提供丰富且好用的大转盘工具，让用户在轻松便捷的操作中即可完成各种选择，还支持复杂场景的自定义配置，真正做到小而美，完美契合微信小程序 “用完即走” 的平台理念。

## 架构设计

![整体架构图](/小程序/架构图.png)

### 前端

- 框架：[Tarojs](https://docs.taro.zone/docs/)，一款支持 React 语法打包到各个平台的小程序跨端框架 [Github 35.3k 🌟](https://github.com/NervJS/taro)
- 状态管理：[Zustand](https://github.com/pmndrs/zustand) ，以功能强大、心智负担小著称的状态管理库，对比 Redux 库显著减少样板代码，它的创始人是前端大神 [dai-shi](https://github.com/dai-shi) ，作者还开发了其他状态管理库（[jotai](https://github.com/pmndrs/jotai)、[valtio](https://github.com/pmndrs/valtio)、[react-tracked](https://github.com/dai-shi/react-tracked)） [Github 45.8k 🌟](https://github.com/pmndrs/zustand)
- 数据请求：[ReactQuery](https://tanstack.com/query/latest/docs/framework/react/overview)，用于在 React 应用中管理数据获取、缓存和同步的强大库，可轻松实现高效的数据请求和状态管理 [Github 41.5k 🌟](https://github.com/tanstack/query)
- 样式方案：[Tailwindcss](https://tailwindcss.com/)，原子化样式方案，通过提供大量的 utility classes，让你可以快速构建响应式、可定制的用户界面，极大地提高了前端开发效率 [Gitbug 81.5k 🌟](https://github.com/tailwindlabs/tailwindcss)
- CI/CD：Github Action，以 github 作为代码管理平台，集成 action 自动化上传/预览小程序

### 后端

- 框架：[Nestjs](https://nestjs.com/) 可以理解为 TypeScript 版的 Spring Boot，使用 DI（依赖注入）方式管理各个模块 [Github 66.5k 🌟](https://github.com/nestjs/nest)
- ORM：[Typeorm](https://typeorm.io/) 一个使用 TypeScript 编写的对象关系映射（ORM）框架，它允许你以面向对象的方式与数据库进行交互，同时支持多种数据库系统 [Github 33.9k 🌟](https://github.com/typeorm/typeorm)
- 数据库：[Mysql](https://www.mysql.com/) 老牌数据库，大小厂都用
- 发布部署: [pm2](https://pm2.keymetrics.io/) 一个守护进程管理器，管理并保持你的应用程序在线 [Gitub 41.4k 🌟](https://github.com/Unitech/pm2)
