## 项目介绍

“小顾的幸运转盘”是一款为广大用户提供方便、快捷的大转盘功能，它主要解决三大场景：

1. 今天不知道吃什么，想要随机点一份套餐
2. 今天不知道做什么运动，想要随机做一下运动
3. 组局吃饭，想要在群聊里随机抽取一位“幸运观众”买单

我们精心提供丰富且好用的大转盘工具，让用户在轻松便捷的操作中即可完成各种选择，还支持复杂场景的自定义配置，真正做到小而美，完美契合微信小程序 “用完即走” 的平台理念。

## 架构设计&技术选型

![整体架构图](/小程序/架构图.png)

### 前端

- 框架：<a href="https://docs.taro.zone/docs/" target="_blank">Tarojs</a>，一款支持 React 语法打包到各个平台的小程序跨端框架 <a href="https://github.com/NervJS/taro" target="_blank">Github <GithubStar repo='taro' owner='nervjs'/>🌟</a>
- 状态管理：<a href="https://github.com/pmndrs/zustand" target="_blank">Zustand</a>，以功能强大、心智负担小著称的状态管理库，对比 Redux 库显著减少样板代码，它的创始人是前端大神 <a href="https://github.com/dai-shi">dai-shi</a>，作者还开发了其他状态管理库（<a href="https://github.com/pmndrs/jotai" target="_blank">jotai</a>、<a href="https://github.com/pmndrs/valtio" target="_blank">valtio</a>、<a href="https://github.com/dai-shi/react-tracked" target="_blank">react-tracked</a>） <a href="https://github.com/pmndrs/zustand" target="_blank">Github <GithubStar owner='pmndrs' repo='zustand'/>🌟</a>
- 数据请求：<a href="https://tanstack.com/query/latest/docs/framework/react/overview" target="_blank">ReactQuery</a>，用于在 React 应用中管理数据获取、缓存和同步的强大库，可轻松实现高效的数据请求和状态管理 <a href="https://github.com/tanstack/query" target="_blank">Github <GithubStar owner='tanstack' repo='query' />🌟</a>
- 样式方案：<a href="https://tailwindcss.com/" target="_blank">Tailwindcss</a>，原子化样式方案，通过提供大量的 utility classes，让你可以快速构建响应式、可定制的用户界面，极大地提高了前端开发效率 <a href="https://github.com/tailwindlabs/tailwindcss" target="_blank">Github <GithubStar owner='tailwindlabs' repo='tailwindcss' />🌟</a>
- CI/CD：Github Action，以 Github 作为代码管理平台，集成 Action 自动化上传/预览小程序

### 后端

- 框架：<a href="https://nestjs.com/" target="_blank">Nestjs</a> 可以理解为 TypeScript 版的 Spring Boot，使用 DI（依赖注入）方式管理各个模块 <a href="https://github.com/nestjs/nest" target="_blank">Github <GithubStar owner='nestjs' repo='nest' />🌟</a>
- ORM：<a href="https://typeorm.io/" target="_blank">Typeorm</a> 一个使用 TypeScript 编写的对象关系映射（ORM）框架，它允许你以面向对象的方式与数据库进行交互，同时支持多种数据库系统 <a href="https://github.com/typeorm/typeorm" target="_blank">Github <GithubStar owner='typeorm' repo='typeorm'/>🌟</a>
- 数据库：<a href="https://www.mysql.com/" target="_blank">Mysql</a> 老牌数据库，大小厂都用
- 发布部署: <a href="https://pm2.keymetrics.io/" target="_blank">PM2</a> 一个守护进程管理器，管理并保持你的应用程序在线 <a href="https://github.com/Unitech/pm2" target="_blank">Github <GithubStar owner='Unitech' repo="pm2" />🌟</a>
